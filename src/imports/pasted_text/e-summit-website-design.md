# MASTER PROMPT — E-SUMMIT WEBSITE (PIRATE × TECH × PREMIUM)

Design an **Awwwards-level premium E-Summit website** with a cinematic storytelling experience. The website should feel like a journey through an entrepreneurial treasure map.

**Theme:** Modern Pirates of the Caribbean × Apple Event × Harvard Business Review × Luxury Editorial Design.

**Mood:** Premium, mysterious, adventurous, futuristic, sophisticated.

### Color Palette

* Deep Navy (#07111F)
* Emerald Green (#0E7A61)
* Antique Gold (#C8A96A)
* Warm Beige Parchment (#E8DCC5)
* Soft White Typography
* Very subtle red accents for hover states

### Visual Style

* Cinematic lighting
* Volumetric fog
* Realistic textures
* No cartoon illustrations
* Minimal but immersive
* Plenty of whitespace
* Glassmorphism only in cards
* Smooth transitions between sections
* Consistent spacing using 12-column grid
* Rounded corners (20–28px)
* Soft shadows and depth

### Theme Elements

Use these subtly throughout the website:

* Compass roses
* Treasure routes
* Ships
* Cliffs
* Lanterns
* Maps
* Anchors
* Ropes
* Ocean waves
* Constellations
* Navigation symbols

Avoid making the design look childish or game-like.
